CPU Scheduling Algorithms
AI-Generated Summary
CPU Scheduling is the process by which the OS decides which process in the ready queue gets CPU time. Key algorithms include:

1. FCFS — First Come First Served. Simplest algorithm, non-preemptive.
2. SJF — Shortest Job First. Minimizes average waiting time.
3. Round Robin — Each process gets a fixed time quantum. Best for time-sharing.
4. Priority Scheduling — Higher priority processes execute first.

Key metric: Minimize waiting time and turnaround time while maximizing CPU utilization.
